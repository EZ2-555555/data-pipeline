# Stale Results (Pre-RRF Hybrid)

Archived evaluation results from the initial hybrid retrieval implementation using **weighted linear scoring** (α/β/γ weights) before the migration to weighted Reciprocal Rank Fusion (RRF) with cross-encoder reranking.

Retained for historical comparison and to document the design evolution described in Chapter 5, Section 5.6 (Sensitivity Analysis) of the final report. The current production results are in `evaluation/results/`.
